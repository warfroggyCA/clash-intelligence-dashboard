## Top 5 Issues

1. **Leadership page is too long** - Break into tabs (Overview, Analytics, Management, Settings)
2. **Missing tooltips** - Add tooltips to all metrics and technical terms
3. **No breadcrumbs** - Add breadcrumb navigation to all pages
4. **War Planning workflow unclear** - Add step indicators or tabs
5. **Inconsistent content hierarchy** - Some pages feel like "things bolted on"

## Quick Wins

- Add tooltips to common metrics (ACE, WPIE, TH, BK, AQ, etc.)
- Add "Back to Top" buttons on long pages
- Standardize card styles and spacing
- Add breadcrumb navigation
- Improve empty state messages

## Design System Gaps

- Tooltip component (standardized)
- Breadcrumb component
- Tab component (for long pages)
- Empty state component
- Metric card component
